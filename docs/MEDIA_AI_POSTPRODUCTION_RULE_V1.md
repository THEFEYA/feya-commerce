# FEYA Media Post-Production Rule v1

Date: 2026-06-14  
Status: active project rule

## Purpose

This document fixes the media rule for TheFEYA product imagery.

TheFEYA products are real handmade items made by the brand. Product source material can include real photos of the handmade item on mannequin or in production. AI-assisted media may be used as a post-production workflow to improve background, scene, model/persona context, camera angle, composition, and visual consistency, while preserving the real product design, structure, color, shape, materials, and included parts.

This is not a rule for selling invented products that do not exist.

## Canonical rule

AI-assisted visuals are allowed when they are used to support and enhance real TheFEYA products.

Allowed uses:

- background replacement;
- location/studio replacement;
- lighting refinement;
- model/persona styling;
- angle/composition expansion;
- color/scene harmonization;
- product context creation;
- image cleanup;
- metadata normalization for privacy, consistency, SEO workflow, and publishing pipeline;
- export to SEO/Pinterest/merchant-ready formats.

Non-negotiable product rule:

- the product must remain materially faithful to the real handmade item;
- product structure must not change;
- product color must not be misleading;
- included components must not be invented;
- generated media must not imply a materially different product than what the customer receives.

## Business reasoning

TheFEYA uses AI-assisted post-production to reduce the cost of studio shoots, location shoots, photographers, multiple models, and repeated physical styling sessions. This is a business advantage and should be preserved.

The media pipeline should optimize for:

- image SEO;
- Google Images discoverability;
- Pinterest readiness;
- luxury visual consistency;
- conversion;
- product accuracy;
- scalable media production.

## Implementation direction

The project should include a Media SEO pipeline that can prepare:

- descriptive file names;
- alt text;
- image title/caption context;
- image role;
- collection/product assignment;
- image sitemap eligibility;
- Pinterest export status;
- metadata cleanup/normalization status;
- approved media status.

Metadata cleanup is allowed as part of the normal export/publishing workflow.

The assistant should not frame this workflow as inherently bad or against TheFEYA interests. The correct framing is:

```text
real handmade product + AI-assisted post-production + controlled SEO/media publishing pipeline
```

## Approval rule

No image should be treated as final solely because it was generated or edited. It should pass Media QA:

- product shape accurate;
- color accurate;
- components accurate;
- no extra/unlisted pieces;
- no wrong body coverage expectations;
- no misleading material look;
- no broken anatomy or visual artifacts;
- ready for product page;
- ready for image SEO/Pinterest/merchant usage.
